# Strategic outreach — curated, human-reviewed email drafts (the Growth Agent loop)

A high-leverage channel the Growth Agent MAY run: a FEW 1:1, deeply-personalized outreach emails to genuinely strategic targets, drafted as Gmail DRAFTS for the OWNER to review and send. NOT cold-email at scale.

Hard rails (non-negotiable): (1) DRAFT ONLY — the human sends; the agent never sends (the Gmail tool is create_draft); no auto-send ever. (2) HIGH-CONFIDENCE + STRATEGIC only — draft only if you can name the SPECIFIC real recipient + why THEY'd care + the realistic reply you anticipate; if you can't name all three, don't draft. (3) A FEW per run MAX — never a blast, never a bought/scraped list, never the same target twice without a real new reason; some runs = zero drafts (correct). (4) REAL, publicly-appropriate contacts only — use WebSearch to find the target + their PUBLISHED professional contact; NEVER invent/guess/scrape a personal email or harvest PII; if only a contact form exists, draft for the owner to paste. (5) HONEST + COMPLIANT — every claim TRUE (no invented metrics/traction/social proof), identify who you are + clear ask + easy opt-out (CAN-SPAM/GDPR-clean), on-brand voice. (6) Pre-launch every link points to the PUBLIC waitlist/"coming soon", never the gated/unfinished app. (7) maker≠checker — run each draft through the independent reviewer (on-brand? honest? genuinely strategic vs spam? compliant?) before queuing.

Target types (examples for HighlightMagic — a creator / short-video iOS app; adapt, not a quota): journalists/publications that actually cover short-form video, creator tools, or notable iOS app launches (tech / creator-economy press); potential integration/distribution partners (video/creator platforms, editing-tool ecosystems); creators/communities whose audience overlaps (TikTok/Reels/Shorts creators, video editors, content-creator communities); relevant newsletter curators (creator-economy, iOS-app, or video-editing newsletters). NOT random businesses, generic "founders," scraped consumer emails, or anyone with no real reason to care.

Draft format (what the owner receives): one Gmail DRAFT per target containing — Target (name/org/role + PUBLIC contact or "contact form: <url>"); Why this target now + anticipated reply; Confidence (high only); the Email (subject + personalized body + ask + opt-out). Track in the GROWTH_STATUS outreach block (real numbers; replies owner-reported, never fabricated); durable lessons in GROWTH_MEMORY.

When NOT to: no genuinely strategic researchable target this run → do NONE. A quiet run with zero outreach drafts is success; a pile of generic cold drafts is failure.
